✅ Creates config directories (~/.claude, ~/.chatit)

✅ Creates global 'chatit' command
✅ Optional API provider setup
✅ Creates startup scripts

⏱️  Takes: 2-3 minutes
🎯 Result: Ready to use immediately with 'chatit' command

  chatit "Your prompt"

  (Completely FREE for free tier models)

Option 3: OpenCode (Local Server - 75+ Providers)
  opencode server
  export OPENCODE_ENDPOINT="http://localhost:4096"
  chatit --provider opencode

"chatit: command not found"
  → Check PATH with: echo $PATH | grep ".local/bin"
  → Or use full path: ~/.chatit/chatit "prompt"

"Node.js 18+ required"
  → Download from https://nodejs.org
  → Or install Bun: curl -fsSL https://bun.sh/install | bash

"API key not working"
  → Verify with: echo $ANTHROPIC_API_KEY
  → Re-authenticate: chatit /login
  → Check diagnostics: chatit /doctor

For more help:
  chatit /help
  cat ~/.chatit/INSTALLATION_GUIDE.md
